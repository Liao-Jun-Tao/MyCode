   - 分解子问题
      讲要排序的数组从中间分割为两半
      然后再将分割出来的每个子数组各分割为两半
      重复以上操作, 只有一个元素为主 return arr
   - 求解每个子问题
      粒度最小的子数组, 两两合并确保有序
   - 合并子问题的解 得出大问题的解
      合并的规模越来越大, 得到一个完全排序的数组
