    js开发中经常需要把一个多维数组一维化,flatten

    1. 遍历+Array.isArray递归! 传统解决
    2. reduce 可以优化命令式,解决更优美,
    []  result  pre 已经扁平化的部分数组, cur Array.isArray 判断 是否递归

      - 闭包 高阶函数 返回值是函数

      - map
            es6 数组的API,基于一个数组每一项执行一个函数,返回组成一个新的数组
